Description détaillée

Classe gérant la détection des formes humaine. Utilise la détection implémenté dans OpenCV avec la méthode detectMultiScale() qui est une implémentation de l'algorithme de Viola & Jones. Hérité de l'interface DetectionProcessing ce qui lui permet d'être utilisé dans le cadre d'un AlgorithmManager.
