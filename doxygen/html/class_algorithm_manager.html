Description détaillée

Classe de gestion des processus. Permet de gérer le lancement des processus qui lui ont été donnée par les méthodes add*(). Gère également le chargement des paramètres contenus dans le fichier INI de configuration et initialise chaque processus dont il a la charge.

void AlgorithmManager::addDecision ( DecisionProcessing *  decision)

Ajoute un processus de décision au manager, celui ci sera exécuté lors de l'appel à decisionRun(). Plusieurs processus de decision peuvent être ajouter par cette méthode, ils seront exécutés dans l'ordre d'ajout.

bool AlgorithmManager::loadParameters ( string  file_name)

Charge les paramètres contenus dansle fichier INI dont le nom est file_name. Le fonctionnement du chargement des paramètres est expliqués dans la documentation fourni avec le programme. Cette fonction n'a pas besoin d'être modifier pour prendre en compte des nouveaux paramètres ou de nouvelles classes, seules les classes des processus nécessitent d'être modifiées (méthode setParameters()).

Paramètres
file_namechaine de caractère correspondant au chemin d'accès du fichier (config.ini par exemple).
